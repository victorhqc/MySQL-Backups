MySQL-Backups
=============

Easily backup all your MySQL databases into a tar file.

##Usage
```
$ python backups.py --mysql ./mysql.cnf --cnf ./config.json
```


##Requirements
* Python
* https://github.com/jbardin/scp.py (Main files already in repo)
